■■ はじめに ■■

online EditorはLinuxとPHPの環境で動作するエディターです。
エディター部分はACEエディターを、デザインはBootstrapを使っています。
１ページWikiとしてPawfalikiを使っています。（ここは全面改装の予定です）

すべて、自己責任でご使用ください。
このソフトウェアを使用し、いかなる損害をうけたとしても、一切の保証はいたしません。

このパッケージ内容物について、複写、移植、改変、転載、再配布、すべて許諾します。


■■ 動作環境 ■■

IA64のCENT OS 6.3上で開発、検証しています。
PHPが動作する環境ならほぼ動くでしょう。
ブラウザはFireFoxを推奨しますが、chromeでも動作します。
他のブラウザでの動作は確認していません。

■■ 使用方法 ■■


■ 実行準備

最低限、wizd.conf の
---------------------
document_root           /               # 動画ファイル置き場
---------------------
を、適切に設定してください。
データが置かれているディレクトリをフルパスで指定します。
document_rootの外にあるデータを、使いたい場合には、document_rootに
シンボリックリンクを作成してご使用ください。



スキンを使用する場合には、最低限以下の３項目を適切に設定してください。
init起動する場合には、skin_rootはフルパスで指定しないと不具合が出ます。
埋め込みスキン（flag_use_skin false)は、まだ未対応です。
-------
# スキン使用するかフラグ
flag_use_skin           true

# スキンデータ置き場
skin_root               ./skin

# スキン名 (skin_rootにあるディレクトリ名)
skin_name               default
-------



■ 設定

linux上でウェブからアクセスできる適当なフォルダに展開します。
編集すべきフォルダはapache:rootの属性にしておきます。
指定フォルダからguest/guestでログインできます。
ID/PWはlogin.phpにハードコーディングされているので、
適宜変更してください。

エラーは/var/log/httpd/error_logを参照しています。
参照のために、viewlogコマンドを作成する必要があります。
sudo.txtにソースがあるのてコンパイルしてください。
viewlogを有効にするには、viewlogコマンドを作成します。
まずvisudoコマンドを使って末尾に以下のソースを挿入します。

apache ALL=(root) NOPASSWD: /[PATH]/viewlog

[PATH]にはviewlogを生成するパスを設定します。

■ 実行方法
実行すると左側にファイル一覧、右側に使い方がでます。
左のソースをクリックすると右側でマルチタブで編集できます。
編集が終わったらセーブアイコンをクリックしてセーブしてください。
エディターはACEエディターなので、普通に検索／置換などが可能です。
キーバインドやテーマはそのうち。。
左上からコマンドが実行できます。
アップロードは現在ツリー表示されているところへのアップロードです。
ダウンロードはウェブ上のデータを現在位置にダウンロードします。
グラフィック表示はイメージ一覧を表示します。
エクスポートはカレントのデータをtgzまたはzip（インストール内容により変化）
でダウンロードします。
メモは単なるメモです。
他にPHPMyAdminや検索などが可能です。
ファイルにカーソルを当てると、更新日付、ファイルサイズなどを表示します。
Wikiはeditすると編集できます。とりあえず動作するだけです。
